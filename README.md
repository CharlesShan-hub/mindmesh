# mindmesh


## markdown syntex contract
* file-link
    * obsidian: `[[path-to-file|filename]]`
      * eg: `[[programming/c++/README|README]]`
    * origin: `[filename](path-to-file)`
      * eg: `(imgname)[programming/c++/imgname]`
* image-link
    * obsidian: `![[path-to-img|imgname]]`
      * eg: `![[programming/c++/README|README]]`
    * origin: `![imgname](path-to-img)`
      * eg: `!(imgname)[programming/c++/imgname]`
